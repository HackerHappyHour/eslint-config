# @h3/eslint-config

The Hacker Happy Hour js style guide

## Installation

```javascript
$ npm install --save-dev @h3/eslint-config
```

## Usage

Add one of the following to your project's `.eslintrc.*` file:

```javascript
{
  extends: "@h3/eslint-config",
  extends: "@h3/eslint-config/yolk"
}
```
